# PerfectCalendar - 개발 환경 요구사항

## 필수 소프트웨어

### Node.js
- 버전: 20.12.0 이상 (LTS 권장)
- 다운로드: https://nodejs.org/

### Expo CLI
- 설치: 프로젝트 내 자동 설치 (npx 사용)
- 글로벌 설치 불필요

### Android Studio
- 버전: 최신 안정 버전
- 다운로드: https://developer.android.com/studio
- 필수 구성요소:
  - Android SDK
  - Android SDK Platform
  - Android Virtual Device (AVD)

### 환경 변수 설정 (Windows)
- ANDROID_HOME: C:\Users\[USERNAME]\AppData\Local\Android\Sdk
- PATH에 추가:
  - %ANDROID_HOME%\platform-tools
  - %ANDROID_HOME%\emulator
  - %ANDROID_HOME%\tools
  - %ANDROID_HOME%\tools\bin

### Xcode
- 버전: 최신 버전
- 다운로드: Mac App Store
- Command Line Tools 설치 필요

### CocoaPods
- 설치: sudo gem install cocoapods

### Expo Go 앱
- Android: Google Play Store에서 "Expo Go" 검색
- iOS: App Store에서 "Expo Go" 검색

## 프로젝트 의존성

아래 패키지들은 package.json에 정의되어 있으며, 
npm install 명령어로 자동 설치됩니다:

### 핵심 프레임워크
- expo: ~51.0.0 (또는 최신 버전)
- react: 18.x
- react-native: 0.74.x
- typescript: ~5.3.0

### Node.js 버전 문제
- nvm (Node Version Manager) 사용 권장
- Windows: nvm-windows
- Mac/Linux: nvm

### Android Studio 인식 안 됨
- 환경 변수 확인
- Android Studio 재설치
- SDK Manager에서 필수 패키지 설치 확인

### Expo 앱 연결 안 됨
- 같은 Wi-Fi 네트워크 사용 확인
- 방화벽 설정 확인
- Metro Bundler 포트(19000) 확인
